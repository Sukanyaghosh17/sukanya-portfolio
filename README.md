# Sukanya Ghosh - Personal Portfolio

A modern, responsive personal portfolio website built to showcase my skills, projects, and professional background. Designed with a sleek, glassmorphism aesthetic to provide an engaging user experience.

## ✨ Features
- **Modern UI/UX**: Clean and minimalist design with glassmorphism elements and smooth micro-animations.
- **Responsive Layout**: Fluid design that seamlessly adapts from desktop down to mobile devices.
- **Custom Sections**:
  - **Hero**: Eye-catching introduction with dynamic gradient typography.
  - **About**: Brief personal overview.
  - **Skills**: Highlighted technical proficiencies.
  - **Education**: Minimalist timeline detailing academic background.
  - **Projects**: Grid display of recent work with descriptive cards.
  - **Contact**: User-friendly form and professional social links.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure.
- **CSS3 / Tailwind CSS**: For rapid UI development, flex/grid layouts, responsive typography, and styling.
- **JavaScript (Vanilla)**: For smooth scrolling, interactivity, and mobile navigation.
- **FontAwesome & Custom SVGs**: Scalable vector icons.

## 🚀 Getting Started

To view the project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/Sukanyaghosh17/PORTFOLIO.git
   ```
2. Navigate to the project directory:
   ```bash
   cd PORTFOLIO
   ```
3. Open `index.html` in your favorite web browser or use a live server extension (like Live Server in VS Code).
